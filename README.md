# Haptics

Repo of my haptic-related works.

Details will be updated after the paper goes public.

# Quick introduction

Motor_control: An Arduino program designed to control the vibration intensity of eight motors connected to pins 3–10.

Magnitude: A Unity3D program that generates vibration patterns and sends voltage commands to the Arduino for controlling motors.

GoDirectGripper: A Python program to read data from Go Direct® Hand Dynamometer(https://www.vernier.com/product/go-direct-hand-dynamometer/)

Force: A Unity3D program that generates vibration patterns based on the data sent from GoDirectGripper
